# ISO Sample Results

This directory contains example output generated by `iso/Review-IsoContents.ps1`.

- `debian-13.5.0-amd64-netinst-file-manifest.csv` was generated from Debian
  13.5.0 amd64 netinst with:

```powershell
.\iso\Review-IsoContents.ps1 `
  -Path .\debian-13.5.0-amd64-netinst.iso `
  -FileChecksums `
  -ChecksumAlgorithm SHA256 `
  -ChecksumCsv .\iso\sample-results\debian-13.5.0-amd64-netinst-file-manifest.csv
```

The CSV lists one ISO-visible file per row with `Path`, `Size`, `Modified`,
`Algorithm`, and `Checksum`.
