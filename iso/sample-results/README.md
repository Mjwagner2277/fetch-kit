# ISO Sample Results

This directory contains example output generated by `iso/Review-IsoContents.ps1`.

- `debian-13.5.0-amd64-netinst-iso-review.csv` was generated from Debian
  13.5.0 amd64 netinst with:

```powershell
.\iso\Review-IsoContents.ps1 `
  -Path .\debian-13.5.0-amd64-netinst.iso `
  -CsvOutput .\iso\sample-results\debian-13.5.0-amd64-netinst-iso-review.csv
```

The CSV uses one wide schema. ISO-visible file rows have `RecordType` =
`ISOFile`; visible RPM metadata rows, when present, have `RecordType` =
`RPMMetadata`. Debian netinst media normally has no directly visible RPM files,
so this sample contains only ISO file rows.
